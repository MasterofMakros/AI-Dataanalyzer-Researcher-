# Product: AI-Ready Data Pool System

## Vision
Transform a 7TB unstructured hard drive into an AI-optimized, Nextcloud-synced data lake that enables instant semantic search, intuitive manual navigation, and automated file organization.

## Users
1. **Primary User**: Owner (Business + Private data management)
2. **AI Agents**: Local RAG systems, Gemini CLI, future automation bots
3. **Nextcloud Clients**: Web, Desktop, Mobile for cross-device access

## Goals
1. **Instant Findability**: Any file findable in <5 seconds via AI query or folder navigation
2. **Zero Duplication**: Deduplicated storage with hash-based integrity
3. **Software Safety**: Preserve runtime dependencies for servers/games
4. **Privacy First**: Local AI indexing, no cloud data leakage
5. **Future-Proof**: GraphRAG-ready structure with semantic context files

## High-Level Features
- [x] Data Pool Taxonomy (01-12, 99 numbering)
- [x] Semantic `_context.md` files per folder
- [x] Git-versioned migration manifest
- [x] AI-Ready naming conventions
- [ ] Automated Inbox sorting workflow
- [ ] Local RAG system integration (AnythingLLM/GPT4All)
- [ ] GraphRAG knowledge graph generation
